Главная цель проекта

Наблюдение, оценка и прогноз изменений законодательства о защите персональных данных, составление технической документации.


Преимущества проекта

Изучение профессии технического писателя, аналитика, контент-менеджера, перспективная и хорошо оплачеваемая работа.


Задание

Есть команда разработки, которая собирается разработать модуль, работающий на русском и английском языках, для выявления на сайтах нарушений и уязвимостей, связанных с формами для ввода персональных данных. Модуль с графическим интерфейсом пользователя, где пользователь может видеть результаты проверки сайта и рекомендуемые действия. Нужно составить техническую документацию по этому проекту.


Техническое задание

Разработано техническое задание в упрощённом виде, соответствуя основным положениям ГОСТа.


Сценарий использования

Написан документ "сценарий использования". Данный документ подразумевает ответы на основные вопросы:

  • кто целевой пользователь?
  • какую проблему он хочет решить?
  • что он для этого сделает?
  • в каком виде получит результат?
